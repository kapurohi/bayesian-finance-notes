# Chapter 12 — Computationally Efficient Markov Chain Simulation

Metropolis-Hastings and Gibbs sampling work but are slow for complex, high-dimensional models. This chapter covers the modern MCMC algorithms that make large Bayesian financial models computationally feasible — specifically Hamiltonian Monte Carlo (HMC) and the No-U-Turn Sampler (NUTS), which is the default in PyMC.
