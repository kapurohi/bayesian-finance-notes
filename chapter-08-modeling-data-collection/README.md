# Chapter 8 — Modeling Accounting for Data Collection

Real data is rarely a clean random sample from the population you care about. Data collection introduces selection effects, truncation, censoring, and stratification. This chapter covers how to build models that account for the data collection process itself — so your inferences reflect the underlying population, not the sample you happened to observe.
