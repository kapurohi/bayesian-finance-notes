# Chapter 9 — Decision Analysis

Bayesian inference produces probability distributions. But at some point, someone has to make a decision — buy or sell, approve or deny, hold or exit. This chapter connects Bayesian posterior distributions to formal decision-making under uncertainty using **expected utility theory**.
