# 9.3 — Decisions with Unknown Utilities

## What this section is doing

In practice, the utility function is rarely known with certainty. Different stakeholders have different risk preferences. Gelman discusses sensitivity analysis over the utility function — how much does the optimal decision change as the utility function varies?

---

## Financial application

**Mandate-driven investing:** A pension fund manager has fiduciary obligations that constrain the effective utility function. A hedge fund manager has a very different utility function. The same posterior distribution over intrinsic value can lead to different optimal actions under different utility functions.

**Sensitivity analysis:** Rather than committing to a single utility function, compute the optimal action across a range of plausible utility parameters. If the same decision is optimal across all plausible utilities, it is **robustly optimal**. If the optimal action changes significantly across utility functions, more information about preferences is needed before committing.
