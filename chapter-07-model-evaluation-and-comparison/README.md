# Chapter 7 — Evaluating, Comparing, and Expanding Models

This chapter covers how to compare competing models and how to expand a model when it fails a check. The central tools are information criteria (WAIC, LOO-CV) for model comparison and model expansion strategies for addressing specific failures identified by posterior predictive checks.
