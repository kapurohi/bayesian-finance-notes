# Chapter 15 — Hierarchical Linear Models

Chapter 14 estimated one set of regression coefficients. Chapter 15 estimates many sets — one per company, group, or time period — with all coefficient sets drawn from a common population distribution. This is the combination of Chapter 5's hierarchical structure and Chapter 14's regression framework.
