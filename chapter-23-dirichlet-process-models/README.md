# Chapter 23 — Dirichlet Process Models

The Dirichlet process (DP) is the nonparametric Bayesian extension of finite mixture models. Instead of fixing the number of mixture components $K$ in advance, the DP allows the number of components to grow with the data — the model learns how many distinct groups exist from the data itself.
