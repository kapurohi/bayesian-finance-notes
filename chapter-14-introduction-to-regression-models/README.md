# Chapter 14 — Introduction to Regression Models

Regression models connect outcomes to predictors through a linear or generalized linear structure. This chapter introduces the Bayesian treatment of regression — what changes compared to OLS, what stays the same, and why the Bayesian version is more useful for financial modeling.
