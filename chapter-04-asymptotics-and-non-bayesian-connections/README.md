# Chapter 4 — Asymptotics and Connections to Non-Bayesian Approaches

This chapter is about what happens to Bayesian posteriors as data grows large — and how Bayesian methods connect to the frequentist methods you may have used before. The core result is that with enough data, Bayesian and frequentist approaches converge to the same answer. This chapter explains why, and what that means for practical modeling.
