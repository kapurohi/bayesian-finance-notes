# Chapter 16 — Generalized Linear Models

Linear regression assumes a continuous Normal outcome. Generalized linear models (GLMs) extend the framework to other outcome types: binary outcomes (logistic regression), counts (Poisson regression), positive continuous outcomes (Gamma regression). This chapter covers Bayesian GLMs — the workhorses of credit modeling and classification.
