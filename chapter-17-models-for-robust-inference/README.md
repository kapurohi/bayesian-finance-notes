# Chapter 17 — Models for Robust Inference

Normal models are sensitive to outliers and heavy tails. This chapter covers models that are robust to these — primarily Student-t likelihoods and contamination mixture models. For financial data, where tail events are common and misspecified tails lead to badly wrong risk estimates, this chapter is directly relevant.
