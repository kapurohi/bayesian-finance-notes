# Chapter 20 — Basis Function Models

Basis function models extend linear regression by transforming predictors through a set of basis functions before entering the linear model. This allows flexible nonlinear relationships to be captured within the linear regression framework. Splines, polynomial expansions, and radial basis functions are all examples.
