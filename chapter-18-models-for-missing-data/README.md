# Chapter 18 — Models for Missing Data

Financial data is always incomplete. Companies report late. Data vendors have gaps. Some metrics are only reported annually. This chapter covers Bayesian approaches to missing data — which treat missing values as unknown parameters to be inferred from the available data.
